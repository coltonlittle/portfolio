Web

Westbeach Plumbing & Heating

This web project was created for the client, Westbeach Plumbing & Heating. The client wanted to establish an online presence, since the company has never had any online components to their business. The client gave me a few specifications to follow: What he wanted the logo to look like, and the site's colour scheme. I created and was responsible for every detail of the website. I designed the site and the logo, and developed it individually. The site is finished, and is just waiting for content creation.

Little House Media - Parallax Scrolling

Ever since I discovered parallax scrolling on the web, I've always wanted to build something using that technique. This page is meant to be a pseudo description of services for a graphic design/new media firm with a contact form at the end. This page was built using the Stellar.js, Waypoints, and jQuery Easing libraries. All images are created/photographed myself, as well as a few images from free image site, Stock.Xchng.

Vinyl Society

Vinyl Society was built for a class project. The criteria was to build a site using a specified layout of two main content columns, beside two sidebars, followed by a main content column beneath them. The home and newsletter pages are the only active pages. Another part of the project was to build a form asking users a set of questions related their music preferences and habits; the form is setup to email the results to the site's admin. All images were supplied by Stock.X.chng.

Emilio Finatti - Interface Design

The purpose of this project was to find a client and create, or improve on, a set of steps in a user interface. I chose to create a user interface for a local pizza place, Emilio Finatti's. I designed an online ordering system for the restaurant. The series of steps for the user in this instance is to order two large pepperoni pizzas, for the current time, for delivery. The gallery goes through the specified process. All images were supplied by Stock.Xchng.
